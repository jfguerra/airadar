Design Guidelines Applied

  • 24×24px artboard following Material Design grid system
  • Four separate L-shaped corner brackets with rounded ends
  • Centered horizontal minus sign for focus/scan indicator
  • 2.5dp stroke width for optimal clarity and modern appearance
  • Rounded stroke caps (linecap="round") for smooth, friendly corners
  • Symmetrical, perfectly centered, and balanced composition
  • Clean, flat design with no gradients, shadows, or outlines
  • Monochrome design using solid black or currentColor for theming
  • Vector-based for perfect scaling at any size
  • Perfect for scan, focus, capture, and frame detection actions
